# Gas Turbine Cycle Simulation & Optimization

## Project Overview
This project involves the comprehensive thermodynamic modeling of advanced gas turbine cycles. Using **EES (Engineering Equation Solver)**, the study evaluates performance metrics for various configurations, including simple Brayton, regenerative, reheating, and intercooling cycles.

## Technical Highlights
* **Cycle Analysis**: Modeled state points for air-standard cycles with variable specific heats.
* **Parametric Studies**: Analyzed the impact of pressure ratios ($r_p$) and turbine inlet temperatures (TIT) on thermal efficiency and net work output.
* **Optimization**: Determined the optimal pressure ratio for multi-stage compression and expansion to maximize overall system performance.

## Tools Used
* **EES (Engineering Equation Solver)**: Used for high-accuracy property lookups and solving non-linear systems of equations.
